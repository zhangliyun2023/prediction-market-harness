# intel/ — 量化情报层

为交易场景设计的情报层:信息的价值 = 它还没被市场定价的程度。
**这一层的存在理由:交易里,情报的价值 = 它还没被市场定价的程度。**

## 三支柱

1. **登录一次,持久化会话,扫描中途绝不刷新**

2. **金子是后端 JSON,拦截页面自己的 XHR,不抓 DOM**
   采集技术阶梯:
   Tier0 DOM(兜底) → **Tier1 响应拦截(默认)** → T2 参数测绘 → T3 页内 fetch → T4 CDP → T5 干净公开 API。
   英文源(HN/StackExchange/Reddit archive)天生 Tier5,`collect_en.py` 直连公开 API。

3. **信号分层 + 跨平台三角验证**:

   | 层 | 定义 | 交易含义 |
   |---|---|---|
   | **P0 已定价** | 盘口价格本身、英文主流媒体头条、已刷屏的消息 | **无 edge。看到新闻再下单 = 接盘** |
   | **P1 半定价** | 中文平台先爆但英文圈未跟进 / 专业小圈子在传 / 官方渠道刚挂出未扩散 | 可能有 edge,取决于扩散速度,分钟级窗口 |
   | **P2 未定价** | 一手观察、你亲验过的事实、地理/语言隔离的局部信息 | **真 edge 候选。这是唯一值得下注的层** |

   **三角验证规则:单一来源不下注。** P2 信号要么两个独立来源互证,要么你能亲验。
   降级规则:一条信号随时间自动降级(P2 → P1 → P0),扩散即贬值——快就是一切。

## 决策闭环(情报 → 钱,每步都有闸门)

```
collect(采集) → evidence.py 分层打标 → 对应哪个盘? (gamma 搜索)
  → P0? 扔掉  → P1/P2? 估 edge → scout.py 查容量(edge×容量才是钱)
  → 值得? 先 paper.py 记纸上单 → 攒够样本才谈真钱(orders.py 有 $20 硬顶)
```

## 反模式

- **互动量 ≠ 声量**:KOL 视频 158 万互动是视频触达,不是事件信号强度。
- **水军检测**:高度模板化+emoji 刷屏的评论区,信号价值为零。
- **同音词/缩写去噪**:搜盘口关键词时先想碰撞词。
- **别信单平台**:每个平台有偏(知乎长文、抖音大众+KOL、微博热点、Reddit 圈层)。
- **别盲目重试限流端点**(尤其 m.weibo.cn),失败请求也计数。
- **(量化)看到英文头条才动 = P0 = 接盘**。英文头条是你的"截止线",不是"信号源"。
- **(量化)情报再好,先过容量检查**:P2 信号 + $300 深度的盘 = 期望值几块钱,别为它熬夜。
